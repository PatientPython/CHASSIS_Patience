## Naming Summary

- Global variable: `G_[Type][Module]_Name`
- Static variable: `S_[Type][Module]_Name`
- Local variable: plain descriptive name
- Struct type: `ST_Xxx`, instance: `st_XX_Xxx`
- Enum type: `EM_Xxx`, values: `XXX_STATE_NAME`
- Macro/constants: uppercase with underscores
- Function: public none, internal `_`, low-level `__`
