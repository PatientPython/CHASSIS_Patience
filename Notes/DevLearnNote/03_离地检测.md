# 第三章 地面支持力估计与离地检测

## 3.1 离地检测的目的

在平衡步兵底盘系统中，准确检测腿部是否离地至关重要：

1. **安全保护**: 离地时禁止某些控制动作，防止失控
2. **控制切换**: 离地状态下使用不同的控制参数（如 LQR 的 K 矩阵调整）
3. **跳跃控制**: 作为跳跃状态机的触发条件

## 3.2 地面支持力计算原理

### 3.2.1 整体思路

地面支持力 `F_N` 通过**动力学分析**计算，而非直接使用力传感器。计算流程如下：

```
关节电机力矩 T1, T4
       ↓
虚功原理逆映射
       ↓
虚拟摆杆力 F_Leg 和力矩 Tp_Leg
       ↓
腿部动力学分析
       ↓
腿对轮子的竖直作用力 P
       ↓
轮子动力学分析
       ↓
地面支持力 F_N
```

### 3.2.2 轮子 Z 轴加速度计算

轮子在 Z 轴方向的加速度由以下项组成：

```c
// API/Algorithm.c L1142
ZAcc_Wheel = ZAcc_Body
           - L0_ddot * cos(Phi)
           + 2 * L0_dot * Phi_dot * sin(Phi)
           + L0 * Phi_ddot * sin(Phi)
           + L0 * Phi_dot² * cos(Phi)
```

**各项物理含义**:
| 项 | 物理含义 |
|----|----------|
| `ZAcc_Body` | 机体 Z 轴加速度（IMU 测量） |
| `-L0_ddot * cos(Phi)` | 摆杆伸缩加速度在 Z 轴的投影 |
| `2*L0_dot*Phi_dot*sin(Phi)` | 科里奥利加速度项 |
| `L0 * Phi_ddot * sin(Phi)` | 摆杆角加速度引起的切向加速度 |
| `L0 * Phi_dot² * cos(Phi)` | 向心加速度在 Z 轴的投影 |

### 3.2.3 地面支持力公式

根据牛顿第二定律：

```
F_N = F_P + G_w + M_w * ZAcc_Wheel
```

其中：
- `F_N`: 地面支持力 (N)
- `F_P`: 腿对轮子的竖直作用力 (N)
- `G_w = M_w * g`: 轮子重力 (N)
- `M_w`: 轮子质量 (kg)
- `ZAcc_Wheel`: 轮子 Z 轴加速度 (m/s²)

## 3.3 离地检测数据结构

### 3.3.1 结构体定义

```c
// API/Algorithm.h
typedef struct {
    // 初始化参数
    float M_w;           // 轮子质量 (kg)
    float M_l;           // 腿部质量 (kg)
    float g;             // 重力加速度 (m/s²)
    float SampleTime;    // 采样时间 (s)

    // 五连杆相关
    float L1, L4;        // 大腿杆长度 (mm)
    float Phi1, Phi4;    // 大腿角度 (度)
    float Phi2, Phi3;    // 小腿角度 (度)
    float Phi0, L0;      // 等效杆角度 (度) 和长度 (mm)
    float Theta;         // 与竖直方向夹角 (度)

    // 运动学变量
    float L0_dot, L0_ddot;    // 等效杆长度变化率和加速度
    float Theta_dot, Theta_ddot; // 摆杆角速度和角加速度
    float Phi, Phi_dot, Phi_ddot; // 地面坐标系下摆杆角度和微分

    // 动力学变量
    float T1, T4;        // 关节电机力矩 (Nm)
    float F_Leg, Tp_Leg; // 虚拟摆杆力和力矩

    // 输出变量
    float ZAcc_Body;     // 机体 Z 轴加速度 (m/s²)
    float ZAcc_Wheel;    // 轮子 Z 轴加速度 (m/s²)
    float F_N;           // 地面支持力 (N)
} OffGround_StructTypeDef;

// 全局变量
OffGround_StructTypeDef GstCH_OffGround1;  // 左腿离地检测
OffGround_StructTypeDef GstCH_OffGround2;  // 右腿离地检测
```

```c
// API/Algorithm.c L1021-1049
void OffGround_LegLinkRelateDataUpdate(LegLinkageCal_StructTypeDef LegPtr,
                                       OffGround_StructTypeDef *pOffGrd,
                                       RobotSide_EnumTypeDef LegSide) {
    // 1. 复制五连杆参数
    pOffGrd->L1 = LegPtr.l1;
    pOffGrd->L4 = LegPtr.l4;
    pOffGrd->Phi1 = LegPtr.phi1;
    pOffGrd->Phi4 = LegPtr.phi4;
    pOffGrd->Phi2 = LegPtr.phi2;
    pOffGrd->Phi3 = LegPtr.phi3;
    pOffGrd->L0 = LegPtr.L0;
    pOffGrd->Phi0 = LegPtr.phi0;

    // 2. 计算 Theta（与竖直方向夹角，地面坐标系）
    float ThetaTemp = LegLinkage_GetTheta(&LegPtr, LegSide);
    pOffGrd->Theta = ThetaTemp;

    // 3. 减去车身俯仰角，得到地面坐标系下的角度
    pOffGrd->Phi = ThetaTemp - pOffGrd->PitchAngle;

    // 4. 计算角速度
    pOffGrd->Phi_dot = LegLinkage_GetThetadot(&LegPtr, LegSide) - pOffGrd->PitchAngleVel;

    // 5. 计算长度变化率和加速度
    pOffGrd->L0_dot = LegLinkage_GetL0dot(&LegPtr);
    // L0_ddot 通过差分计算
    pOffGrd->L0_ddot = (pOffGrd->L0_dot - pOffGrd->L0_dot_Pre) / pOffGrd->SampleTime;
    pOffGrd->L0_dot_Pre = pOffGrd->L0_dot;

    // 6. 计算角加速度（差分）
    pOffGrd->Phi_ddot = (pOffGrd->Phi_dot - pOffGrd->Phi_dot_Pre) / pOffGrd->SampleTime;
    pOffGrd->Phi_dot_Pre = pOffGrd->Phi_dot;
}
```

### 3.4.2 核心计算函数

**计算虚拟摆杆的力和力矩**:
```c
// API/Algorithm.c L1071-1101
void OffGround_GetRealFAndTp(OffGround_StructTypeDef *pOffGrd) {
    // 1. 计算 VMC 雅可比矩阵的逆矩阵
    float J_Matrix_Inverse[2][2];

    float L1 = pOffGrd->L1, L4 = pOffGrd->L4, L0 = pOffGrd->L0;
    float Phi1 = pOffGrd->Phi1, Phi2 = pOffGrd->Phi2;
    float Phi3 = pOffGrd->Phi3, Phi4 = pOffGrd->Phi4;
    float Phi0 = pOffGrd->Phi0;

    // 中间变量
    float Sigma1 = L4 * (cos(Phi0-Phi2)*sin(Phi0-Phi3)*sin(Phi3-Phi4)
                       - cos(Phi0-Phi3)*sin(Phi0-Phi2)*sin(Phi3-Phi4));
    float Sigma2 = L1 * (cos(Phi0-Phi2)*sin(Phi0-Phi3)*sin(Phi1-Phi2)
                       - cos(Phi0-Phi3)*sin(Phi0-Phi2)*sin(Phi1-Phi2));
    float Sigma3 = sin(Phi2 - Phi3);

    // 逆矩阵元素
    J_Matrix_Inverse[0][0] = -cos(Phi0-Phi2) * Sigma3 / Sigma2;
    J_Matrix_Inverse[0][1] =  cos(Phi0-Phi3) * Sigma3 / Sigma1;
    J_Matrix_Inverse[1][0] =  L0 * sin(Phi0-Phi2) * Sigma3 / Sigma2;
    J_Matrix_Inverse[1][1] = -L0 * sin(Phi0-Phi3) * Sigma3 / Sigma1;

    // 2. 计算真实的 F 和 Tp
    float T1 = pOffGrd->T1, T4 = pOffGrd->T4;
    pOffGrd->F_Leg  = J_Matrix_Inverse[0][0] * T1 + J_Matrix_Inverse[0][1] * T4;
    pOffGrd->Tp_Leg = J_Matrix_Inverse[1][0] * T1 + J_Matrix_Inverse[1][1] * T4;
}
```

**计算腿对轮子的竖直作用力**:
```c
// API/Algorithm.c L1109-1120
float OffGround_GetLegToWheelForce(OffGround_StructTypeDef *pOffGrd) {
    float G_l = pOffGrd->M_l * pOffGrd->g;  // 腿部重力

    float Phi = pOffGrd->Phi;    // 地面坐标系下摆杆与竖直方向夹角
    float F  = pOffGrd->F_Leg;   // 沿杆方向的力
    float Tp = pOffGrd->Tp_Leg;  // 绕杆的力矩
    float L0 = pOffGrd->L0;      // 摆杆长度

    // F_P = G_l + F*cos(Phi) + Tp*sin(Phi)/L0
    float F_P = G_l + F * cos(Phi) + Tp * sin(Phi) / L0;
    return F_P;
}
```

**计算地面支持力**:
```c
// API/Algorithm.c L1128-1147
float OffGround_GetSupportForce(OffGround_StructTypeDef *pOffGrd) {
    // 1. 获取腿对轮子的竖直作用力
    float F_P = OffGround_GetLegToWheelForce(pOffGrd);

    // 2. 轮子重力
    float G_w = pOffGrd->M_w * pOffGrd->g;

    // 3. 轮子 Z 轴加速度
    float ZAcc_Wheel = ZAcc_Body
                     - L0_ddot * cos(Phi)
                     + 2 * L0_dot * Phi_dot * sin(Phi)
                     + L0 * Phi_ddot * sin(Phi)
                     + L0 * Phi_dot * Phi_dot * cos(Phi);

    // 4. 地面支持力 = 腿对轮子作用力 + 轮子重力 + 轮子惯性力
    pOffGrd->F_N = F_P + G_w + pOffGrd->M_w * ZAcc_Wheel;

    return pOffGrd->F_N;
}
```

## 3.5 离地状态判断

### 3.5.1 迟滞比较器

为防止在临界值附近频繁切换状态，采用**迟滞比较器**：

```c
// Application/Chassis/Chassis_APIFunction.c L95-106
bool _Ch_OffGroundStateFlagGet(bool F_OffGroundPre, float LegF_N) {
    bool F_OffGroundNext = F_OffGroundPre;

    if (F_OffGroundPre == false && LegF_N <= 25.0f) {
        // 未离地 → 离地：支持力 <= 25N
        F_OffGroundNext = true;
    }
    else if (F_OffGroundPre == true && LegF_N >= 70.0f) {
        // 离地 → 未离地：支持力 >= 70N
        F_OffGroundNext = false;
    }

    return F_OffGroundNext;
}
```

**迟滞参数**:
| 参数 | 值 | 说明 |
|------|-----|------|
| 离地阈值 | 25N | 支持力低于此值判定为离地 |
| 落地阈值 | 70N | 支持力高于此值判定为落地 |
| 迟滞区间 | 45N | 防止状态抖动 |

### 3.5.2 离地状态主处理函数

```c
// Application/Chassis/Chassis_APIFunction.c L451-477
void CH_OffGround_Process(void) {
    /* 左腿离地检测 */
    OffGround_BodyZAccUpdate(&GstCH_OffGround1, GSTCH_Data.AccZFB);
    OffGround_PitchAngleUpdate(&GstCH_OffGround1, GSTCH_Data.PitchAngleFB);
    OffGround_PitchAngleVelUpdate(&GstCH_OffGround1, GSTCH_Data.PitchAngleVelFB);
    OffGround_LegLinkRelateDataUpdate(GstCH_LegLinkCal1, &GstCH_OffGround1, LeftSide);
    OffGround_TorqueDataUpdate(&GstCH_OffGround1,
                               GSTCH_JM3.TorqueFB,  // 左腿前关节
                               GSTCH_JM1.TorqueFB); // 左腿后关节
    OffGround_GetRealFAndTp(&GstCH_OffGround1);

    /* 右腿离地检测 */
    OffGround_BodyZAccUpdate(&GstCH_OffGround2, GSTCH_Data.AccZFB);
    OffGround_PitchAngleUpdate(&GstCH_OffGround2, GSTCH_Data.PitchAngleFB);
    OffGround_PitchAngleVelUpdate(&GstCH_OffGround2, GSTCH_Data.PitchAngleVelFB);
    OffGround_LegLinkRelateDataUpdate(GstCH_LegLinkCal2, &GstCH_OffGround2, RightSide);
    OffGround_TorqueDataUpdate(&GstCH_OffGround2,
                               GSTCH_JM2.TorqueFB,  // 右腿前关节
                               GSTCH_JM4.TorqueFB); // 右腿后关节
    OffGround_GetRealFAndTp(&GstCH_OffGround2);

    /* 支持力计算和状态判断 */
    float Leg1F_N_RawValue = OffGround_GetSupportForce(&GstCH_OffGround1);
    float Leg2F_N_RawValue = OffGround_GetSupportForce(&GstCH_OffGround2);

    // 低通滤波
    GSTCH_Data.Leg1F_N = _Ch_FBData_LPF(Leg1F_N_RawValue, &GstCH_Leg1F_N_LPF);
    GSTCH_Data.Leg2F_N = _Ch_FBData_LPF(Leg2F_N_RawValue, &GstCH_Leg2F_N_LPF);

    // 状态判断（迟滞比较）
    GSTCH_Data.F_OffGround1 = _Ch_OffGroundStateFlagGet(GSTCH_Data.F_OffGround1,
                                                         GSTCH_Data.Leg1F_N);
    GSTCH_Data.F_OffGround2 = _Ch_OffGroundStateFlagGet(GSTCH_Data.F_OffGround2,
                                                         GSTCH_Data.Leg2F_N);
}
```
