# 第六章 状态机与遥控器逻辑

## 6.1 概述

平衡步兵底盘采用**层次化状态机**进行模式管理，实现不同控制策略之间的平滑切换。状态机的输入来自遥控器指令、传感器状态和通讯状态。

## 6.2 底盘模式定义

### 6.2.1 模式枚举

```c
// Global/GlobalDeclare_Chassis.h
typedef enum {
    CHMode_RC_ManualSafe = 0,    // 手动安全模式
    CHMode_RC_AutoSafe,          // 自动安全模式
    CHMode_RC_Standby,           // 待机模式
    CHMode_RC_StandUp,           // 起立模式
    CHMode_RC_Free,              // 自由模式
    CHMode_RC_Follow,            // 跟随模式
    CHMode_RC_SitDown,           // 坐下模式
    CHMode_RC_OffGround,         // 离地模式
    CHMode_RC_Jump,              // 跳跃模式
} ChassisMode_EnumTypeDef;
```

### 6.2.2 模式说明

| 模式 | 说明 | 触发条件 |
|------|------|----------|
| ManualSafe | 手动安全，遥控器急停 | 左拨杆在下且右拨杆在上 |
| AutoSafe | 自动安全，通讯故障保护 | 遥控器/电机/IMU 通讯异常 |
| Standby | 待机，准备启动 | 从安全模式恢复 |
| StandUp | 起立，从坐姿站起 | 拨轮上拨 |
| Free | 自由模式，非跟随运动 | 起立完成/左拨杆回中 |
| Follow | 跟随模式，跟随云台 | 左拨杆在上 |
| SitDown | 坐下，收纳腿长 | 拨轮上拨 |
| OffGround | 离地保护 | 离地检测触发 |
| Jump | 跳跃 | 拨轮下拨 |

## 6.3 遥控器指令解析

### 6.3.1 遥控器连接检测

```c
// API/General_AuxiliaryFunc.c L39-72
bool IsRCConnected(void) {
    // 1. 检查串口帧率
    if (GST_SystemMonitor.USART1Rx_fps < USART1Rx_fpsMinTH) {
        return false;  // 帧率过低，未连接
    }

    // 2. 检查通道值范围
    if (GST_Receiver.ST_RC.JoyStickL_X < RCChannelValue_Min ||
        GST_Receiver.ST_RC.JoyStickL_X > RCChannelValue_Max) {
        return false;  // 通道值异常
    }
    // ... 检查其他通道

    // 3. 检查拨杆位置有效性
    if (GST_Receiver.ST_RC.Level_L == RCLevel_Up ||
        GST_Receiver.ST_RC.Level_L == RCLevel_Mid ||
        GST_Receiver.ST_RC.Level_L == RCLevel_Down) {
        return true;  // 左拨杆有效
    }
    if (GST_Receiver.ST_RC.Level_R == RCLevel_Up ||
        GST_Receiver.ST_RC.Level_R == RCLevel_Mid ||
        GST_Receiver.ST_RC.Level_R == RCLevel_Down) {
        return true;  // 右拨杆有效
    }

    return false;  // 默认未连接
}
```

## 6.4 状态机架构

### 6.4.1 三层架构

状态机采用**三层优先级架构**：

```
┌─────────────────────────────────┐
│ Layer 1: 全局中断 (Interrupts)  │  ← 最高优先级
│  - ManualSafe (左下右上)        │
│  - AutoSafe (通讯故障)          │
└──────────────┬──────────────────┘
               │
┌──────────────▼──────────────────┐
│ Layer 2: 状态恢复 (Recovery)    │  ← 中优先级
│  - ManualSafe → Standby         │
│  - AutoSafe → Standby           │
└──────────────┬──────────────────┘
               │
┌──────────────▼──────────────────┐
│ Layer 3: 核心流程 (Core Flow)   │  ← 正常操作
│  - Standby ↔ Free ↔ Follow     │
│  - Free/Follow → Jump          │
│  - Free/Follow → OffGround     │
└─────────────────────────────────┘
```

### 6.4.2 状态机主函数

```c
// Application/Chassis/Chassis_Stratgy.c L204-299
ChassisMode_EnumTypeDef ChassisStrategy_ModeChoose_RCControl(
    Chassis_ModeChooseParameter_StructTypeDef ST_ModeChoosePara) {

    ChassisMode_EnumTypeDef CurrentMode = ST_ModeChoosePara.MC_ModePre;
    ChassisMode_EnumTypeDef NextMode = CurrentMode;

    // 记忆跳跃/离地前的活跃模式
    static ChassisMode_EnumTypeDef LastActiveMode = CHMode_RC_Free;

    /* ========================================================== */
    /* Layer 1: 全局中断                                            */
    /* ========================================================== */

    // 正常 → ManualSafe: 左下右上
    if (_IsTrans_Normal_To_ManualSafe()) {
        return CHMode_RC_ManualSafe;
    }

    // 正常 → AutoSafe: 通讯异常
    if (_IsTrans_Normal_To_AutoSafe(ST_ModeChoosePara)) {
        return CHMode_RC_AutoSafe;
    }

    /* ========================================================== */
    /* Layer 2 & 3: 状态机流转                                      */
    /* ========================================================== */

    switch (CurrentMode) {
        // --- 故障与待机 ---
        case CHMode_RC_ManualSafe:
            if (_IsTrans_ManualSafe_To_Standby())
                NextMode = CHMode_RC_Standby;
            break;

        case CHMode_RC_AutoSafe:
            if (_IsTrans_AutoSafe_To_Standby(ST_ModeChoosePara))
                NextMode = CHMode_RC_Standby;
            break;

        case CHMode_RC_Standby:
            if (_IsTrans_Cmd_StandUp())
                NextMode = CHMode_RC_StandUp;
            break;

        // --- 启动与关闭 ---
        case CHMode_RC_StandUp:
            if (_IsTrans_StandUp_To_Free(ST_ModeChoosePara)) {
                NextMode = CHMode_RC_Free;
                LastActiveMode = CHMode_RC_Free;
            }
            break;

        case CHMode_RC_SitDown:
            if (_IsTrans_SitDown_To_Standby(ST_ModeChoosePara))
                NextMode = CHMode_RC_Standby;
            break;

        // --- 核心运动模式 (Free) ---
        case CHMode_RC_Free:
            LastActiveMode = CHMode_RC_Free;  // 更新记忆

            if (_IsTrans_Cmd_SitDown())
                NextMode = CHMode_RC_SitDown;
            else if (_IsTrans_Free_To_Follow())
                NextMode = CHMode_RC_Follow;
            else if (_IsTrans_Cmd_Jump())
                NextMode = CHMode_RC_Jump;
            else if (_IsTrans_To_OffGround(ST_ModeChoosePara))
                NextMode = CHMode_RC_OffGround;
            break;

        // --- 核心运动模式 (Follow) ---
        case CHMode_RC_Follow:
            LastActiveMode = CHMode_RC_Follow;  // 更新记忆

            if (_IsTrans_Follow_To_Free())
                NextMode = CHMode_RC_Free;
            else if (_IsTrans_Cmd_Jump())
                NextMode = CHMode_RC_Jump;
            else if (_IsTrans_To_OffGround(ST_ModeChoosePara))
                NextMode = CHMode_RC_OffGround;
            break;

        // --- 特殊动作 (Jump) ---
        case CHMode_RC_Jump:
            // 落地后恢复到 LastActiveMode
            if (GSTCH_Data.F_JumpLanding && _IsTrans_To_Landed(ST_ModeChoosePara)) {
                NextMode = LastActiveMode;
            }
            break;

        // --- 特殊状态 (OffGround) ---
        case CHMode_RC_OffGround:
            // 落地后恢复到 LastActiveMode
            if (_IsTrans_To_Landed(ST_ModeChoosePara)) {
                NextMode = LastActiveMode;
            }
            break;
    }

    return NextMode;
}
```

## 6.5 状态转移条件

### 6.5.1 全局中断转移

**ManualSafe 触发**:
```c
// L60-67
bool _IsTrans_Normal_To_ManualSafe(void) {
    // 左拨杆在下 且 右拨杆在上
    return IsLeftLevelDown() && IsRightLevelUp();
}
```

**AutoSafe 触发**:
```c
// L69-89
bool _IsTrans_Normal_To_AutoSafe(Chassis_ModeChooseParameter_StructTypeDef ST_ModeChoosePara) {
    // 遥控器断开
    if (IsRCConnected() == false) return true;

    // 轮毂电机通讯异常
    if (HM1_Rx_fps < HubMotorRx_fpsMinTH || HM2_Rx_fps < HubMotorRx_fpsMinTH) return true;

    // IMU 通讯异常
    if (UART4_Rx_fps < UART4Rx_fpsMinTH) return true;

    return false;
}
```

### 6.5.2 状态恢复转移

**ManualSafe → Standby**:
```c
// L95-103
bool _IsTrans_ManualSafe_To_Standby(void) {
    // 左拨杆回中
    return IsLeftLevelMid();
}
```

**AutoSafe → Standby**:
```c
// L105-112
bool _IsTrans_AutoSafe_To_Standby(Chassis_ModeChooseParameter_StructTypeDef ST_ModeChoosePara) {
    // 通讯恢复正常（取 AutoSafe 触发的反）
    return _IsTrans_Normal_To_AutoSafe(ST_ModeChoosePara) == false;
}
```

### 6.5.3 核心流程转移

**Standby → StandUp**:
```c
// L118-126
bool _IsTrans_Cmd_StandUp(void) {
    // 拨轮上拨并回正（锁存信号）
    if (GSTCH_Data.F_RollerUpLatched == true) {
        GSTCH_Data.F_RollerUpLatched = false;  // 清除锁存
        return true;
    }
    return false;
}
```

**StandUp → Free**:
```c
// L128-138
bool _IsTrans_StandUp_To_Free(Chassis_ModeChooseParameter_StructTypeDef ST_ModeChoosePara) {
    float TimeNow = ST_ModeChoosePara.MC_TimeNow;
    float ThisModeStartTime = ST_ModeChoosePara.MC_ST_ModeStartTime.RC_StandUp;
    float ThisModeTotalTime = TimeNow - ThisModeStartTime;

    // 起立时间达到设定值
    return (ThisModeTotalTime >= CHMode_RC_StandUp_TotalTime);
}
```

**Free → Follow**:
```c
// L163-166
bool _IsTrans_Free_To_Follow(void) {

    return IsLeftLevelUp();
}
```

**Follow → Free**:
```c
// L168-171
bool _IsTrans_Follow_To_Free(void) {
    // 左拨杆回中

}
```

**Jump 触发**:

## 6.6 模式更新流程

### 6.6.1 模式参数更新

```c
// Application/Chassis/Chassis_Stratgy.c L28-52
void ChassisStrategy_ModeChooseParaStructUpdate(
    Chassis_ModeChooseParameter_StructTypeDef* pModeChoosePara) {

    // 1. 更新上次模式
    pModeChoosePara->MC_ModePre = GEMCH_ModePre;

    // 2. 更新模式开始时间和当前时间
    pModeChoosePara->MC_ST_ModeStartTime = GSTCH_Data.ST_ModeStartTime;
    pModeChoosePara->MC_TimeNow = RunTimeGet();

    // 3. 更新运动姿态
    pModeChoosePara->MC_LegLenAvgFB = (GSTCH_Data.LegLen1FB + GSTCH_Data.LegLen2FB) / 2.0f;
    pModeChoosePara->MC_VelFB = GSTCH_Data.VelFB;

    // 4. 更新离地标志
    if (GSTCH_Data.F_OffGround1 || GSTCH_Data.F_OffGround2)
        pModeChoosePara->MC_F_OffGround = true;
    else
        pModeChoosePara->MC_F_OffGround = false;

    // 5. AutoSafe 专用变量
    pModeChoosePara->MC_HubMotor1Rx_fps = GST_SystemMonitor.HubMotor1Rx_fps;
    pModeChoosePara->MC_HubMotor2Rx_fps = GST_SystemMonitor.HubMotor2Rx_fps;
    pModeChoosePara->MC_UART4Rx_fps = GST_SystemMonitor.UART4Rx_fps;
}
```

### 6.6.2 模式切换主流程

```c
// Application/Chassis/Chassis_Stratgy.c L342-351
void CH_ChassisModeUpdate(void) {
    Chassis_ModeChooseParameter_StructTypeDef ST_ModeChoosePara_tmp;

    // 1. 保存上次模式
    GEMCH_ModePre = GEMCH_Mode;

    // 2. 更新模式选择参数
    ChassisStrategy_ModeChooseParaStructUpdate(&ST_ModeChoosePara_tmp);

    // 3. 执行状态机，获取新模式
    GEMCH_Mode = ChassisStrategy_ModeChoose_RCControl(ST_ModeChoosePara_tmp);

    // 4. 更新模式开始时间
    ChassisStrategy_ModeStartTimeUpdate(&GSTCH_Data, GEMCH_Mode, GEMCH_ModePre);
}
```

## 6.8 调试建议

| 变量 | 含义 |
|------|------|
| `GEMCH_Mode` | 当前底盘模式 |

```
0: ManualSafe
1: AutoSafe
2: Standby
3: StandUp
4: Free
5: Follow
6: SitDown
7: OffGround
8: Jump
```
