# 第二章 五连杆运动学与动力学解算

## 2.1 五连杆机构概述

### 2.1.1 机械结构

平衡步兵底盘采用**五连杆机构**作为腿部结构，其组成如下：

```
        车身 (Body)
          │
    ┌─────┴─────┐
    │           │
  杆 1 (l1)   杆 4 (l4)
    │           │
  关节 B      关节 D
    │           │
  杆 2 (l2)   杆 3 (l3)
    │           │
    └─────┬─────┘
          │
        关节 C (末端/轮子)
    ──────┴──────
        地面
```

**五连杆参数**:
| 杆号 | 变量 | 长度 (mm) | 描述 |
|------|------|-----------|------|
| 1 | l1 | 135.0 | 左大腿（关节电机 1 到小腿连接点） |
| 2 | l2 | 135.0 | 左小腿（左大腿到轮子中心） |
| 3 | l3 | 135.0 | 右小腿（右大腿到轮子中心） |
| 4 | l4 | 135.0 | 右大腿（关节电机 4 到小腿连接点） |
| 5 | l5 | 170.0 | 两髋关节间距（关节电机 1 到 4 的水平距离） |

### 2.1.2 坐标系定义

五连杆解算使用**局部坐标系**：
- **原点**: 左髋关节 (A 点)
- **X 轴**: 水平向右（从左侧看向右侧）
- **Y 轴**: 竖直向下

**角度定义**:
- `phi1`: 杆 1 与 X 轴正方向夹角（从 X 轴逆时针为正）
- `phi4`: 杆 4 与 X 轴正方向夹角
- `phi2`: 杆 2 与 X 轴正方向夹角
- `phi3`: 杆 3 与 X 轴正方向夹角
- `phi0`: 等效连杆与水平线夹角
- `Theta`: 等效连杆与**竖直方向**的夹角（向后摆为正）

## 2.2 正运动学求解

### 2.2.1 已知条件

**输入**:
- 关节电机 1 角度 `JMAngle1` (度)
- 关节电机 4 角度 `JMAngle4` (度)
- 关节电机零点 `phi1ZP`, `phi4ZP` (度)

**五连杆各点坐标**:
```c
// API/Algorithm.c L391-436
void LegLinkage_ForwardKinematicsCal(LegLinkageCal_StructTypeDef* LegPtr) {
    // 1. 计算各点坐标
    LegPtr->xA = 0;
    LegPtr->yA = 0;
    LegPtr->xE = LegPtr->l5;
    LegPtr->yE = 0;

    // B 点坐标（左髋关节通过杆 1 连接）
    LegPtr->xB = LegPtr->xA + LegPtr->l1 * MyCos(LegPtr->phi1);
    LegPtr->yB = LegPtr->yA + LegPtr->l1 * MySin(LegPtr->phi1);

    // D 点坐标（右髋关节通过杆 4 连接）
    LegPtr->xD = LegPtr->xE + LegPtr->l4 * MyCos(LegPtr->phi4);
    LegPtr->yD = LegPtr->yE + LegPtr->l4 * MySin(LegPtr->phi4);

    // 2. 计算 BD 距离
    float BD = sqrt((xB - xD)² + (yB - yD)²);

    // 3. 计算 phi2（左小腿角度）
    float A1 = 2 * l2 * (xD - xB);
    float B1 = 2 * l2 * (yD - yB);
    float C1 = BD²;
    float Temp1 = sqrt(A1² + B1² - C1²);
    LegPtr->phi2 = 2 * R2A * MyAtan((B1 + Temp1) / (A1 + C1));

    // 4. 计算 phi3（右小腿角度）
    float A2 = 2 * l3 * (xB - xD);
    float B2 = 2 * l3 * (yB - yD);
    float C2 = BD²;
    float Temp2 = sqrt(A2² + B2² - C2²);
    LegPtr->phi3 = 2 * R2A * MyAtan((B2 - Temp2) / (A2 + C2));

    // 5. 计算 C 点坐标（轮子中心）
    LegPtr->xC = LegPtr->xB + LegPtr->l2 * MyCos(LegPtr->phi2);
    LegPtr->yC = LegPtr->yB + LegPtr->l2 * MySin(LegPtr->phi2);

    // 6. 计算等效摆杆 L0 和 phi0
    float Half_l5 = LegPtr->l5 / 2.0f;
    LegPtr->L0 = sqrt((xC - Half_l5)² + yC²);
    LegPtr->phi0 = R2A * MyAtan(yC / (xC - Half_l5));
}
```

### 2.2.2 输出结果

**直接输出**:
| 变量 | 含义 | 单位 |
|------|------|------|
| `xC`, `yC` | 轮子中心坐标 | mm |
| `phi2`, `phi3` | 小腿角度 | 度 |
| `L0` | 等效摆杆长度 | mm |
| `phi0` | 等效摆杆与水平线夹角 | 度 |

**腿长定义**:
```c
// API/Algorithm.c L446-448
float LegLinkage_GetLegLength(LegLinkageCal_StructTypeDef* LegPtr) {
    return LegPtr->yC;  // 竖直高度，单位 mm
}
```

> **注意**: 这里的"腿长"是指关节电机到地面的**竖直距离** `yC`，而非等效杆长 `L0`。

## 2.3 运动学微分：速度与角速度

### 2.3.1 关节角速度输入

```c
// API/Algorithm.c L377-381
void LegLinkage_AngleVelDataUpdate(LegLinkageCal_StructTypeDef* LegPtr,
                                   float JMAngleVel1, float JMVelValue2) {
    LegPtr->phi1_dot = JMAngleVel1;  // 直接赋值，单位：度/s
    LegPtr->phi4_dot = JMVelValue2;
}
```

### 2.3.2 C 点 X 方向速度

C 点 X 坐标的微分通过**差分法**计算：

```c
// API/Algorithm.c L334-335, L502-511
// 在 ForwardKinematicsCal 中保存上次值并计算微分
LegPtr->xC_dot = (LegPtr->xC - LegPtr->xC_Pre) / LegPtr->SampleTime;
LegPtr->xC_Pre = LegPtr->xC;

// 获取函数（转换单位并考虑左右方向）
float LegLinkage_GetxCdot(LegLinkageCal_StructTypeDef* LegPtr,
                          RobotSide_EnumTypeDef LegSide) {
    if (LegSide == LeftSide) {
        return LegPtr->xC_dot * MM2M;  // mm/s → m/s
    } else {
        return -LegPtr->xC_dot * MM2M;  // 右腿方向相反
    }
}
```

### 2.3.3 Theta 角速度：雅可比矩阵方法

**为什么不用数值微分？**

对 `Theta` 直接进行差分会导致：
1. **噪声放大**: 小角度变化被采样时间除法放大
2. **毛刺问题**: 临界位置可能出现跳变

**解决方案**: 使用**运动学雅可比矩阵**，直接由关节角速度计算 `Theta_dot`：

```c
// API/Algorithm.c L553-589
float LegLinkage_GetThetadot(LegLinkageCal_StructTypeDef* LegPtr,
                             RobotSide_EnumTypeDef LegSide) {
    // 计算雅可比矩阵元素
    // J12 = L1*cos(phi0-phi3)*sin(phi1-phi2) / [L0*sin(phi3-phi2)]
    float J_Matrix12 = LegPtr->l1 * MM2M *
                       MyCos(LegPtr->phi0 - LegPtr->phi3) *
                       MySin(LegPtr->phi1 - LegPtr->phi2) /
                       (LegPtr->L0 * MM2M * MySin(LegPtr->phi3 - LegPtr->phi2));

    // J22 = L4*cos(phi0-phi2)*sin(phi3-phi4) / [L0*sin(phi3-phi2)]
    float J_Matrix22 = LegPtr->l4 * MM2M *
                       MyCos(LegPtr->phi0 - LegPtr->phi2) *
                       MySin(LegPtr->phi3 - LegPtr->phi4) /
                       (LegPtr->L0 * MM2M * MySin(LegPtr->phi3 - LegPtr->phi2));

    // 防止分母为零导致 NaN
    if (isnan(J_Matrix12)) J_Matrix12 = 0.0f;
    if (isnan(J_Matrix22)) J_Matrix22 = 0.0f;

    // Theta_dot = J12 * phi1_dot + J22 * phi4_dot
    float Theta_dot_Temp = J_Matrix12 * LegPtr->phi1_dot +
                           J_Matrix22 * LegPtr->phi4_dot;

    // 根据左右侧确定正方向
    if (LegSide == LeftSide) {
        return Theta_dot_Temp;
    } else {
        return -Theta_dot_Temp;  // 右腿方向相反
    }
}
```

**雅可比矩阵的物理意义**:
```
[Theta_dot]   [J12  J22]   [phi1_dot]
[    L0_dot ] = [J11  J21] * [phi4_dot]
```

其中：
- `J12`: phi1 角速度对 Theta 角速度的传递系数
- `J22`: phi4 角速度对 Theta 角速度的传递系数

### 2.3.4 L0 长度变化率

```c
// API/Algorithm.c L519-536
float LegLinkage_GetL0dot(LegLinkageCal_StructTypeDef* LegPtr) {
    // 雅可比矩阵元素（用于计算 L0_dot）
    float T_Matrix11 = LegPtr->l1*MM2M *
                       MySin(LegPtr->phi0-LegPtr->phi3) *
                       MySin(LegPtr->phi1-LegPtr->phi2) /
                       MySin(LegPtr->phi3-LegPtr->phi2);

    float T_Matrix21 = LegPtr->l4*MM2M *
                       MySin(LegPtr->phi0-LegPtr->phi2) *
                       MySin(LegPtr->phi3-LegPtr->phi4) /
                       MySin(LegPtr->phi3-LegPtr->phi2);

    // 防止 NaN
    if (isnan(T_Matrix11)) T_Matrix11 = 0.0f;
    if (isnan(T_Matrix21)) T_Matrix21 = 0.0f;

    // L0_dot = T11 * phi1_dot + T21 * phi4_dot
    return (T_Matrix11 * LegPtr->phi1_dot + T_Matrix21 * LegPtr->phi4_dot) * MM2M;
}
```

## 2.4 Theta 角的获取

### 2.4.1 Theta 角定义

`Theta` 是等效连杆与**竖直方向**的夹角，定义为：
- **正方向**: 向后摆动（轮子相对于髋关节向后）
- **单位**: 度

### 2.4.2 计算公式

```c
// API/Algorithm.c L474-489
float LegLinkage_GetTheta(LegLinkageCal_StructTypeDef* LegPtr,
                          RobotSide_EnumTypeDef LegSide) {
    float ThetaTemp;
    switch (LegSide) {
        case LeftSide:
            // 左腿：Theta = -atan((xC - l5/2) / yC)
            ThetaTemp = -MyAtan((LegPtr->xC - LegPtr->l5/2.0f) / LegPtr->yC) * R2A;
            break;
        case RightSide:
            // 右腿：Theta = +atan((xC - l5/2) / yC)
            ThetaTemp = MyAtan((LegPtr->xC - LegPtr->l5/2.0f) / LegPtr->yC) * R2A;
            break;
    }
    return ThetaTemp;
}
```

**几何解释**:
```
           竖直方向
              │
              │
        髋关节│
          ╲   │
           ╲  │ Theta
            ╲ │
             ╲│
              C (轮子)
```

```
tan(Theta) = (xC - l5/2) / yC
```

## 2.5 VMC 虚功原理与力映射

### 2.5.1 虚功原理简介

虚功原理指出：对于理想约束系统，平衡时所有虚位移做的虚功之和为零。

在五连杆系统中，虚功原理用于将**虚拟摆杆**的力和力矩映射到**关节电机**的力矩：

```
T_joint = J(q)ᵀ * F_virtual
```

其中：
- `T_joint`: 关节电机力矩向量 `[T1, T4]ᵀ`
- `J(q)`: 雅可比矩阵（与位姿 `q` 相关）
- `F_virtual`: 虚拟力和力矩 `[F_along_leg, Torque_about_leg]ᵀ`

### 2.5.2 VMC 数据结构

```c
// API/Algorithm.h
typedef struct {
    float J_Matrix[2][2];  // 雅可比矩阵 T = J * F
    float F_Matrix[2];     // 虚拟力和力矩 [F_along, Torque]
    float T_Matrix[2];     // 关节电机力矩 [T1, T4]
} VMC_StructTypeDef;
```

### 2.5.3 VMC 力矩阵更新

```c
// API/Algorithm.c L892-905
void VMC_FMatrixUpdate(VMC_StructTypeDef* VMCptr, float Force, float Torque,
                       RobotSide_EnumTypeDef LegSide) {
    // 沿杆方向的力
    VMCptr->F_Matrix[0] = Force;

    // 力矩（注意左右腿方向）
    if (LegSide == LeftSide) {
        VMCptr->F_Matrix[1] = Torque;  // 左腿：直接使用
    } else {
        VMCptr->F_Matrix[1] = -Torque;  // 右腿：取反，保证向后转为正
    }
}
```

### 2.5.4 VMC 核心计算：力矩映射

```c
// API/Algorithm.c L919-933
void VMC_Cal(VMC_StructTypeDef* VMCptr, LegLinkageCal_StructTypeDef* LegPtr) {
    // 1. 计算雅可比矩阵
    // J[0][0] = L1*sin(phi0-phi3)*sin(phi1-phi2) / sin(phi3-phi2)
    VMCptr->J_Matrix[0][0] = LegPtr->l1 * MM2M *
        MySin(LegPtr->phi0 - LegPtr->phi3) *
        MySin(LegPtr->phi1 - LegPtr->phi2) /
        MySin(LegPtr->phi3 - LegPtr->phi2);

    // J[0][1] = L1*cos(phi0-phi3)*sin(phi1-phi2) / (L0*sin(phi3-phi2))
    VMCptr->J_Matrix[0][1] = LegPtr->l1 * MM2M *
        MyCos(LegPtr->phi0 - LegPtr->phi3) *
        MySin(LegPtr->phi1 - LegPtr->phi2) /
        (LegPtr->L0 * MM2M * MySin(LegPtr->phi3 - LegPtr->phi2));

    // J[1][0] = L4*sin(phi0-phi2)*sin(phi3-phi4) / sin(phi3-phi2)
    VMCptr->J_Matrix[1][0] = LegPtr->l4 * MM2M *
        MySin(LegPtr->phi0 - LegPtr->phi2) *
        MySin(LegPtr->phi3 - LegPtr->phi4) /
        MySin(LegPtr->phi3 - LegPtr->phi2);

    // J[1][1] = L4*cos(phi0-phi2)*sin(phi3-phi4) / (L0*sin(phi3-phi2))
    VMCptr->J_Matrix[1][1] = LegPtr->l4 * MM2M *
        MyCos(LegPtr->phi0 - LegPtr->phi2) *
        MySin(LegPtr->phi3 - LegPtr->phi4) /
        (LegPtr->L0 * MM2M * MySin(LegPtr->phi3 - LegPtr->phi2));

    // 2. 力矩映射：T = J * F
    VMCptr->T_Matrix[0] = VMCptr->J_Matrix[0][0] * VMCptr->F_Matrix[0] +
                          VMCptr->J_Matrix[0][1] * VMCptr->F_Matrix[1];
    VMCptr->T_Matrix[1] = VMCptr->J_Matrix[1][0] * VMCptr->F_Matrix[0] +
                          VMCptr->J_Matrix[1][1] * VMCptr->F_Matrix[1];
}
```

**映射关系**:
```
[T1]   [J00  J01]   [F_along]
[T ] = [J   J  ] * [T       ]
  4     10   11     virtual
```

展开为：
```
T1 = J00 * F_along + J01 * T_virtual
T4 = J10 * F_along + J11 * T_virtual
```

### 2.5.5 关节电机力矩读取

```c
// API/Algorithm.c L948-954
float VMC_Get_TMatrix(VMC_StructTypeDef* VMCptr, int index) {
    if (index < 0 || index > 1) {
        return 0.0f;
    }
    return VMCptr->T_Matrix[index];
}
```

**返回值**:
| index | 返回 | 对应关节 |
|-------|------|----------|
| 0 | T1 | 关节电机 1 (phi1) |
| 1 | T4 | 关节电机 4 (phi4) |

## 2.6 API 封装与调用

### 2.6.1 数据结构定义

```c
// API/Algorithm.h
typedef struct {
    // 连杆参数（初始化时赋值）
    float l1, l2, l3, l4, l5;     // 五连杆各杆长度 (mm)
    float phi1ZP, phi4ZP;          // 关节电机编码器零点 (度)
    float SampleTime;              // 采样时间 (s)

    // 计算结果（运行时更新）
    float xA, xB, xC, xD, xE;      // 各连接点 X 坐标
    float yA, yB, yC, yD, yE;      // 各连接点 Y 坐标
    float phi1, phi4, phi2, phi3;  // 各杆夹角 (度)
    float L0, phi0;                // 等效连杆长度 (mm) 和角度 (度)
    float phi1_dot, phi4_dot;      // 关节角速度 (度/s)
    float xC_Pre, xC_dot;          // C 点 X 坐标上次值和微分 (mm/s)
} LegLinkageCal_StructTypeDef;

// 全局变量
LegLinkageCal_StructTypeDef GstCH_LegLinkCal1;  // 左腿五连杆解算
LegLinkageCal_StructTypeDef GstCH_LegLinkCal2;  // 右腿五连杆解算
```

### 2.6.2 主处理函数

```c
// Application/Chassis/Chassis_APIFunction.c L412-442
void CH_LegKinematics_Process(void) {
    /* 左腿五连杆解算 */
    // 1. 更新角度数据（关节电机角度 + 零点）
    LegLinkage_AngleDataUpdate(&GstCH_LegLinkCal1,
                               GSTCH_JM1.AngleFB,  // 左腿前侧关节电机
                               GSTCH_JM2.AngleFB); // 左腿后侧关节电机

    // 2. 更新角速度数据
    LegLinkage_AngleVelDataUpdate(&GstCH_LegLinkCal1,
                                  GSTCH_JM1.AngleVelFB,
                                  GSTCH_JM2.AngleVelFB);

    // 3. 执行正运动学计算
    LegLinkage_ForwardKinematicsCal(&GstCH_LegLinkCal1);

    /* 右腿五连杆解算 */
    LegLinkage_AngleDataUpdate(&GstCH_LegLinkCal2,
                               GSTCH_JM3.AngleFB,
                               GSTCH_JM4.AngleFB);
    LegLinkage_AngleVelDataUpdate(&GstCH_LegLinkCal2,
                                  GSTCH_JM3.AngleVelFB,
                                  GSTCH_JM4.AngleVelFB);
    LegLinkage_ForwardKinematicsCal(&GstCH_LegLinkCal2);
}
```

### 2.7.1 关键观测变量

### 2.7.2 奇点位置

五连杆机构在以下位置可能出现**奇点**（分母为零）：

```
sin(phi3 - phi2) = 0  →  phi3 = phi2 或 phi3 = phi2 + 180°
```

这对应于**小腿共线**的情况。代码中已做保护：

```c
if (isnan(J_Matrix12)) J_Matrix12 = 0.0f;
if (isnan(J_Matrix22)) J_Matrix22 = 0.0f;
```
