# 平衡步兵底盘控制系统 - 技术文档

## 概述

本文档详细介绍 RoboMaster 第 26 赛季平衡步兵底盘控制系统的完整技术实现。该系统基于 STM32F407VETx 微控制器，运行 FreeRTOS 实时操作系统，实现了一个自平衡五连杆轮腿式机器人的完整控制方案。

### 系统核心特点

- **控制频率**: 1kHz 主控制回路
- **核心算法**: LQR 平衡控制 + VMC 虚拟模型控制
- **传感器融合**: 轮毂电机编码器 + 关节电机编码器 + IMU
- **执行机构**: 2 个轮毂电机 (C620 电调) + 4 个关节电机 (大疆 M2006)

### 坐标系定义

| 轴向 | 正方向定义 |
|------|------------|
| Pitch (俯仰) | 抬头为正 (+) |
| Yaw (偏航) | 俯视逆时针为正 (+) |
| Roll (横滚) | 左侧高为正 (+) |

### 整体控制架构

```
┌─────────────────────────────────────────────────────────────┐
│                    ChassisTask (1kHz)                       │
├─────────────────────────────────────────────────────────────┤
│  1. CH_FBData_Parse()        ← 传感器反馈数据解析           │
│  2. CH_LegKinematics_Process() ← 腿部五连杆正运动学         │
│  3. CH_OffGround_Process()   ← 离地检测与支持力估计         │
│  4. CH_VelKF_Process()       ← 卡尔曼滤波速度观测           │
│  5. CH_HMTorqueComp_Process() ← 轮毂电机力矩补偿            │
│  6. CH_RCInputPre_Process()  ← 遥控器指令预处理             │
│  7. CH_ChassisModeUpdate()   ← 状态机模式选择               │
│  8. ChassisModeControl_RCControl() ← 模式控制执行           │
└─────────────────────────────────────────────────────────────┘
```

### 数据流管道

系统采用 **Parse → Process → Distribute** 三段式数据流模式：

1. **Parse (解析)**: 原始传感器数据 (CAN/UART 中断) 解析为辅助结构体 (前缀 `st`)
2. **Process (处理)**: 在 `ChassisTask` 中执行滤波、运动学、动力学、控制律计算
3. **Distribute (分发)**: 更新形式化结构体 (前缀 `ST`) 供上层逻辑使用

> **重要约定**: 上层代码**只能**使用形式化结构体 (`ST` 前缀) 中的数据。

### 命名规范说明

**变量前缀规则**: `[Scope][Type][Component]_[Name]`

| 前缀 | 含义 | 示例 |
|------|------|------|
| `G` | 全局 (Global) | `GSTCH_Data` |
| `S` | 静态 (Static) | `s_K_Matrix` |
| `ST` | 形式化结构体 | `GSTCH_Data` |
| `st` | 辅助结构体 | `GstCH_LQRCal` |
| `EM` | 枚举 | `GEMCH_Mode` |
| `F` | 标志位 | `GFCH_SafeMode` |
| `CH` | 底盘组件 | `CH_LQRCal_Process` |
| `HM` | 轮毂电机 | `HM_DesDataUpdate` |
| `JM` | 关节电机 | `JM_DesDataUpdate` |

**标准后缀**:
- `FB`: 反馈值/实际值 (Feedback)
- `Des`: 目标值/期望值 (Desired)
- `ZP`: 零点校准 (Zero Point)

## 快速开始

### API 调用示例

```c
// 1. 腿部运动学解算
CH_LegKinematics_Process();  // 执行五连杆正运动学

// 2. 获取末端角速度
float theta_dot = LegLinkage_GetThetadot(&GstCH_LegLinkCal1);

// 3. LQR 计算
CH_LQRCal_Process();  // 更新状态向量、K 矩阵、执行计算
float wheel_torque = LQR_Get_uVector(&GstCH_LQRCal, 0);  // 左轮扭矩

// 4. VMC 计算
CH_VMCCal_Process();  // 执行虚拟模型控制

// 5. 离地检测
CH_OffGround_Process();  // 计算支持力、更新离地标志
if (GSTCH_Data.F_OffGround) {
    // 离地状态处理
}

// 6. 速度融合
CH_VelKF_Process();  // 卡尔曼滤波融合轮速和 IMU
float fused_vel = GSTCH_Data.VelXFB;  // 融合后的车身速度
```
